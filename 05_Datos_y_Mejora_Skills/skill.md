# 📊 Skills: Observabilidad y Datos

Competencias para asegurar que el negocio aprenda de su propia operación.

* **Curación de Datasets para RAG:** Habilidad para limpiar y organizar la información histórica del negocio para alimentar la memoria de la IA.
* **Análisis de Negative Feedback:** Identificación de patrones en los errores de la IA para ajustar las instrucciones de raíz.
* **Etiquetado Automático:** Uso de modelos para categorizar y organizar los datos generados diariamente por el negocio.
* **Sincronización Vectorial:** Mantener la base de datos de conocimientos (Pinecone/Weaviate) actualizada en tiempo real con nuevas transacciones.
