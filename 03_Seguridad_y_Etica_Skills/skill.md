# 🛡️ Skills: Seguridad y Ética Operativa

Competencias para proteger la integridad y legalidad del sistema autónomo.

* **Prompt Injection Defense:** Diseño de capas de seguridad que detecten intentos de manipulación de las instrucciones del agente.
* **Gobernanza de Datos Propios:** Implementación de políticas de retención para cumplir con leyes (GDPR) sin afectar la memoria del agente.
* **Auditoría de Sesgos:** Realización de pruebas de "caja negra" para asegurar que las decisiones de la IA sean justas y legales.
* **Gestión de Kill Switches:** Diseño de protocolos de emergencia para detener el sistema si detecta comportamientos anómalos.
