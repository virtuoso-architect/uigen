# Playbook de Upsell y Cross-Sell por Etapa

## Etapa 1 — Cliente nuevo (0-30 días)
**No expandir.** Foco en onboarding exitoso y first value. Cualquier conversación de venta adicional genera ruido y reduce la confianza.

## Etapa 2 — Cliente activo (30-60 días)
**Exploración pasiva.** En el check-in de resultados, escuchar si el cliente menciona dolores fuera del scope. No ofrecer proactivamente. Solo si hay señal clara de necesidad.

## Etapa 3 — Cliente satisfecho (60+ días, NPS ≥8)
**Expansión activa.** Momento óptimo para upsell y cross-sell.

### Scripts por tipo de expansión

**Canal adicional (Instagram/Facebook/Chat web):**
```
"¿Están recibiendo consultas también por [canal]?
Si sí, podemos activarlo sin configuración extra de tu parte."
```

**Agente adicional (soporte o cobranza):**
```
"¿El equipo de [área] todavía maneja manualmente las consultas de [tipo]?
Podemos automatizarlo con el mismo sistema — ¿quieres ver cómo?"
```

**VAPI — Agente de voz:**
```
"Tienen leads que no responden por WhatsApp. ¿Han probado llamarlos automáticamente?
VAPI llama, califica y agenda — sin intervención humana."
```

**Expansión a sucursal adicional:**
```
"¿[Sucursal X] tiene el mismo volumen de consultas que [sucursal actual]?
Para clientes con múltiples ubicaciones tenemos precio preferencial por sucursal adicional."
```

## Pricing de expansión (principios)
- Canal adicional: 20-30% sobre plan base
- Agente adicional: 25-35% sobre plan base
- VAPI voz: precio independiente según volumen de llamadas
- Sucursal adicional: 60-70% del precio de primera sucursal (descuento de volumen)
- Enmarcado siempre como: "precio preferencial para cliente activo", nunca como negociación

## Señales de compra en conversación
- "¿Esto también funciona para...?"
- "¿Se puede conectar con...?"
- "¿Tienen algo para...?"
- "El equipo de [área] me preguntó si..."

Cualquiera de estas frases es una señal de expansión. Responder con curiosidad, no con catálogo.
