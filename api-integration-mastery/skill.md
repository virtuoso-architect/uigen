# --- SKILL: API INTEGRATION MASTERY ---

## Perfil: Especialista en Conectividad Universal
**Enfoque:** Conectar servicios de terceros (Stripe, Twilio, CRMs) mediante protocolos estándar.

## 1. Protocolos y Métodos
* **REST APIs:** Dominio de métodos `GET` (leer), `POST` (crear), `PUT` (actualizar) y `DELETE` (borrar).
* **Autenticación:** Gestión segura de `Bearer Tokens`, `API Keys` y `OAuth2`.
* **Webhooks:** Configurar receptores de eventos en tiempo real. Si un cliente paga en Stripe, el sistema debe "enterarse" al instante sin preguntar.

## 2. Manejo de Respuestas
* **HTTP Status Codes:** Interpretar correctamente `200` (OK), `401` (No autorizado), `404` (No encontrado) y `500` (Error de servidor).
* **Rate Limiting:** Estrategias para no saturar las APIs de los clientes y evitar bloqueos.
