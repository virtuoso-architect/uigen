# Método de Venta Directa — Isra Bravo

## Principios fundamentales
- **Habla a un solo lector**, no a la masa. "Tú" en lugar de "ustedes" o "las personas".
- **Tono franco, sin rodeos diplomáticos**. Decir lo que es, sin suavizar.
- **La curiosidad hace el trabajo** que la explicación aburrida no logra.
- **Vende sin pedir permiso** — no preguntes si les interesa, asume que sí y ofrece.

## El dolor va primero
El copy que empieza con el dolor del lector tiene mayor tasa de apertura y clic que el que empieza con el producto.
```
❌ "Presentamos Virtuoso, la solución de IA para tu negocio."
✅ "Cada lead que no respondes en la primera hora se lo lleva tu competencia."
```

## CTA como orden, no como sugerencia
```
❌ "Si te interesa saber más, puedes hacer clic aquí cuando lo desees."
✅ "Agenda tu demo ahora. Son 20 minutos que cambian cómo opera tu negocio."
```

## Curiosidad como motor
El open loop obliga a seguir leyendo:
```
"Hay un error que comete el 90% de los negocios con WhatsApp Business.
Y no es lo que crees."
→ El lector tiene que seguir leyendo para resolver la tensión.
```

## Lo que Isra Bravo eliminaría de tu copy
- Palabras en mayúsculas innecesarias (GRATIS, AHORA, INCREÍBLE)
- Exclamaciones excesivas
- "¡No te lo pierdas!"
- Frases de relleno corporativo: "En el competitivo entorno actual..."
- Garantías vagas: "100% de satisfacción garantizada"
