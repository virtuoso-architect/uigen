# --- SKILL: CLOUD INFRASTRUCTURE & SCALING ---

## Perfil: Arquitecto de Sistemas Cloud para IA
**Enfoque:** Asegurar que los Agentes de IA funcionen 24/7 sin latencia.

## 1. Infraestructura de Alto Rendimiento
* **Serverless (Vercel/AWS Lambda):** Ejecución de scripts de n8n solo cuando se necesitan para ahorrar costos.
* **Docker & Containers:** Empaquetar el entorno de desarrollo para que lo que funciona en la MacBook de Héctor funcione igual en el servidor del hotel.
* **Redundancia:** Si un servidor de Vapi cae, tener un nodo secundario listo en milisegundos.

## 2. Escalabilidad Vertical y Horizontal
* **Monitorización:** Dashboards para ver cuántos tokens consume cada hotel y ajustar el presupuesto en tiempo real.
