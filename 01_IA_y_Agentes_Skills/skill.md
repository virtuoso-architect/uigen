# 🧠 Skills: IA y Arquitectura de Agentes

Competencias técnicas para construir el motor cognitivo del negocio.

* **Diseño de Grafos de Decisión:** Capacidad para estructurar flujos donde la IA elige entre diferentes caminos lógicos (no lineales).
* **Inyección de Contexto Dinámico:** Habilidad para filtrar qué información específica del negocio debe ir en el prompt para no saturar la ventana de contexto.
* **Evaluación de Modelos (Benchmarking):** Saber medir qué modelo (Gemini, GPT, Claude, Llama) es más eficiente basado en latencia y costo.
* **Orquestación Multi-Agente:** Habilidad para hacer que un "Agente Ejecutor" y un "Agente Auditor" colaboren para refinar resultados.
