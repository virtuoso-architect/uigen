# --- SKILL: VALUE STREAM MAPPING (VSM) ---
**Mentes Maestras:** James Womack, Daniel Jones.
**Aplicación:** Dibujamos el mapa de cómo fluye la información desde que el cliente busca en Google hasta que paga. 
**Detección:** Si el flujo se detiene porque un recepcionista no contestó el teléfono, ahí es donde inyectamos Vapi para "desatascar" el flujo de valor.
